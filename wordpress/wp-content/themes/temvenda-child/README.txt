TEM VENDA Child Theme (para Twenty Twenty-Four)
===============================================

1) Pré-requisito
   - Tema pai instalado e ativo: Twenty Twenty-Four (twentytwentyfour).

2) Instalação
   - No WordPress > Aparência > Temas > Adicionar novo > Enviar tema.
   - Envie o arquivo temvenda-child.zip e ative o tema "TEM VENDA Child".

3) Criar a página de landing
   - Páginas > Adicionar nova > Título: Landing (ou Home).
   - No painel à direita, em "Atributos da página" > "Modelo" selecione: TEM VENDA Landing.
   - Publique a página.

4) Definir como página inicial (opcional)
   - Configurações > Leitura > Sua página inicial exibe > Uma página estática.
   - Selecione a página criada (Landing).

5) Customizações
   - Edite conteúdo/links (WhatsApp, e-mail, cidade) diretamente no arquivo page-landing.php no child theme,
     via Aparência > Editor de arquivos de tema, ou via FTP.
   - Para remover o Tailwind CDN e usar build local, ajuste o template para enfileirar seu CSS gerado.

Suporte rápido
--------------
Se algo quebrar após ativar: volte ao tema anterior, edite com calma e reenvie.
